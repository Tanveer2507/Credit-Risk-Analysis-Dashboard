# 💳 Credit Risk Analysis Dashboard

# 📌 Overview

An interactive Credit Risk Analysis Dashboard built using Machine Learning + Streamlit to predict whether a customer is a high-risk or low-risk loan applicant.

This project simulates a real-world loan approval system used by banks and financial institutions.

# 🎯 Key Features
📊 Interactive and user-friendly dashboard
🤖 Machine Learning model (Random Forest)
🔮 Real-time prediction based on user input
📈 Model performance report
🎨 Modern UI with custom styling
⚡ Fast execution using caching
